🔢 The Indian Number System
India uses a unique system — lakhs and crores instead of millions and billions. 1 lakh = 1,00,000 and 1 crore = 1,00,00,000!
1 Crore = 100 Lakhs = 10,000,000

✨ Did you know?
💻 Python was named after a comedy show!
Guido van Rossum named Python after Monty Python's Flying Circus — not the snake! It's one of the world's most popular programming languages.

Your first Python program

Type this in Python and press Run:

print("Hello, World!")
print("My name is Ustad")
print(2026)

Python reads top to bottom and runs each line one at a time.
